Hour of Code

Hour of Code - This program is designed to demystify code and show that anybody can learn the basics. This is a beginner workshop for Geeklings who have no experience with programming. Every kid should have the opportunity to learn computer science. The Hour of Code is a global movement reaching tens of millions of students in 180+countries, and I am bringing it right here to Code Camp. No experience necessary! Bring a computer or tablet if you want to follow along and give it a try. (For ages 8 and up)
